# Story 3.5: Geração de Template HTML/Knockout.js

## Status

Ready for Review

## Story

**As a** dev agent,
**I want** gerar o template HTML + CSS + JS/Knockout.js a partir do mapeamento de campos,
**so that** o usuário receba um template web funcional que reproduz o layout do PDF com bindings dinâmicos para os dados.

## Acceptance Criteria

1. `TemplateGenerator` recebe `List[FieldMapping]` e `List[TextBlock]` e retorna `html`, `css`, `js`, `exemplo`
2. `html` contém template com bindings Knockout.js: `data-bind="text: campo"` para cada campo mapeado
3. `css` preserva layout visual do PDF: fontes, tamanhos, posicionamento aproximado via `position: absolute`
4. `js` (base.js) contém `ko.applyBindings({ campo: ko.observable('') })` para todos os campos + normalizadores de formato BR
5. `exemplo` contém dados de exemplo em JS (`var exampleData = { campo: "valor exemplo" }`) para preview
6. `fidelityScore` calculado (0–100) comparando campos mapeados com confiança ≥ 0.8 vs total de campos
7. `fidelityComment` é string descritiva: ex. "87% dos campos mapeados com alta confiança"
8. `POST /api/preview` recebe `{ jobId }` e retorna HTML completo (template + dados exemplo injetados) para renderizar em iframe
9. Pipeline completo integrado: após matching (pct 60%), chama TemplateGenerator e emite SSE `step: "Gerando template"`, `pct: 85`, depois `done`
10. `GET /api/result/{jobId}` retorna `ExtractionResult` com todos os campos preenchidos (não mais stubs)

### TemplateGenerator — Referência

```python
# backend/services/template_generator.py
from typing import List
from models.field_mapping import FieldMapping
from models.text_block import TextBlock

class TemplateGenerator:
    def generate(
        self,
        fields: List[FieldMapping],
        blocks: List[TextBlock]
    ) -> dict:
        html = self._generate_html(fields, blocks)
        css = self._generate_css(blocks)
        js = self._generate_js(fields)
        exemplo = self._generate_exemplo(fields)
        return {"html": html, "css": css, "js": js, "exemplo": exemplo}

    def _generate_html(self, fields, blocks) -> str:
        field_names = {f.field_name for f in fields}
        items = []
        for b in blocks:
            if b.text.strip() in field_names or any(
                b.text.strip() in f.field_name for f in fields
            ):
                # É um campo → gerar binding
                field = next((f for f in fields if f.field_name in b.text), None)
                if field:
                    items.append(
                        f'<span class="field" '
                        f'style="left:{b.x*100:.1f}%;top:{b.y*100:.1f}%;" '
                        f'data-bind="text: {field.field_name}"></span>'
                    )
            else:
                # É texto estático
                items.append(
                    f'<span class="label" '
                    f'style="left:{b.x*100:.1f}%;top:{b.y*100:.1f}%;">'
                    f'{b.text}</span>'
                )
        content = "\n".join(items)
        return f"""<!DOCTYPE html>
<html lang="pt-BR">
<head><meta charset="UTF-8"><link rel="stylesheet" href="styles.css"></head>
<body>
<div id="form-container">
{content}
</div>
<script src="https://cdnjs.cloudflare.com/ajax/libs/knockout/3.5.1/knockout-min.js"></script>
<script src="base.js"></script>
</body>
</html>"""

    def _generate_js(self, fields) -> str:
        observables = "\n".join(
            f"  {f.field_name}: ko.observable('{f.pdf_value}'),"
            for f in fields
        )
        return f"""var ViewModel = function() {{
{observables}
}};
ko.applyBindings(new ViewModel());"""

    def _generate_css(self, blocks) -> str:
        rules = []
        rules.append("#form-container { position: relative; width: 100%; min-height: 1123px; }")
        rules.append(".label { position: absolute; font-size: 12px; color: #333; }")
        rules.append(".field { position: absolute; font-size: 12px; color: #000; border-bottom: 1px solid #999; min-width: 80px; }")
        for i, b in enumerate(blocks):
            if b.block_type == "text":
                rules.append(
                    f".block-{i} {{ left: {b.x*100:.1f}%; top: {b.y*100:.1f}%; "
                    f"font-size: {max(b.font_size, 10):.0f}px; }}"
                )
        return "\n".join(rules)

    def _generate_exemplo(self, fields) -> str:
        pairs = ", ".join(
            f'"{f.field_name}": "{f.pdf_value}"'
            for f in fields
        )
        return f"var exampleData = {{{pairs}}};"
```

### FidelityScorer — Referência

```python
# backend/services/fidelity_scorer.py
from typing import List
from models.field_mapping import FieldMapping

class FidelityScorer:
    def score(self, fields: List[FieldMapping]) -> dict:
        if not fields:
            return {"score": 0, "comment": "Nenhum campo mapeado.", "suggestions": []}

        high_conf = [f for f in fields if f.confidence >= 0.8]
        low_conf = [f for f in fields if f.confidence < 0.5]
        score = round(len(high_conf) / len(fields) * 100)

        comment = (
            f"{score}% dos campos mapeados com alta confiança. "
            f"{len(fields) - len(high_conf)} campos requerem revisão."
        )

        suggestions = [
            {"field": f.field_name, "suggestion": "Verificar mapeamento manual",
             "confidence": f.confidence}
            for f in low_conf
        ]

        return {"score": score, "comment": comment, "suggestions": suggestions}
```
