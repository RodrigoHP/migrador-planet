# Story 3.4: Jobs Assíncronos e SSE Progress

## Status

Ready for Review

## Story

**As a** dev agent,
**I want** implementar fila de jobs assíncrona com streaming de progresso via Server-Sent Events,
**so that** o frontend possa iniciar o pipeline de migração e acompanhar o progresso em tempo real sem polling.

## Acceptance Criteria

1. `POST /api/jobs` cria um job, dispara pipeline em background e retorna `{ "jobId": "uuid" }` imediatamente (não bloqueia)
2. `GET /api/progress/{jobId}` é um endpoint SSE que emite eventos enquanto o job processa
3. Eventos SSE emitidos: `step` (string), `pct` (0–100), `done` (sem dados), `error` (com mensagem), `warning` (com mensagem)
4. Formato de cada evento SSE: `event: {tipo}\ndata: {json}\n\n` conforme spec EventSource
5. Jobs gerenciados em memória via dict global: `{ jobId: { status, result, queue } }`
6. Pipeline completo: extração PDF → matching campos → (stub geração template) → `done`
7. Timeout de job: 5 minutos — após timeout emitir `event: error` e limpar estado
8. `GET /api/result/{jobId}` retorna `ExtractionResult` completo após job estar `done`, ou 404 se não encontrado / 409 se ainda processando
9. Múltiplos clientes podem escutar o mesmo `jobId` simultaneamente via SSE
10. Frontend consegue acompanhar progresso completo: 0% → 20% → 60% → 85% → 100% (done)

### JobManager — Referência

```python
# backend/services/job_manager.py
import asyncio, uuid
from typing import Dict, List
from models.job import Job

jobs: Dict[str, Job] = {}
job_queues: Dict[str, List[asyncio.Queue]] = {}

def create_job(job_id: str) -> Job:
    job = Job(job_id=job_id, status="pending")
    jobs[job_id] = job
    job_queues[job_id] = []
    return job

async def emit(job_id: str, event: str, data: dict):
    for q in job_queues.get(job_id, []):
        await q.put({"event": event, "data": data})

def subscribe(job_id: str) -> asyncio.Queue:
    q: asyncio.Queue = asyncio.Queue()
    job_queues.setdefault(job_id, []).append(q)
    return q

def unsubscribe(job_id: str, q: asyncio.Queue):
    if job_id in job_queues:
        try:
            job_queues[job_id].remove(q)
        except ValueError:
            pass  # já removido
```

### Endpoint SSE — Referência

```python
# backend/routers/progress.py
from fastapi import APIRouter, HTTPException
from sse_starlette.sse import EventSourceResponse
import asyncio, json
from services import job_manager

router = APIRouter()

@router.get("/progress/{job_id}")
async def progress(job_id: str):
    if job_id not in job_manager.jobs:
        raise HTTPException(status_code=404, detail="Job não encontrado")

    async def event_generator():
        q = job_manager.subscribe(job_id)
        try:
            while True:
                msg = await asyncio.wait_for(q.get(), timeout=300)
                yield {"event": msg["event"], "data": json.dumps(msg["data"])}
                if msg["event"] in ("done", "error"):
                    break
        except asyncio.TimeoutError:
            yield {"event": "error", "data": json.dumps({"message": "Timeout"})}
        finally:
            job_manager.unsubscribe(job_id, q)

    return EventSourceResponse(event_generator())
```
