### Nota sobre SaveFilePickerOptions / OpenFilePickerOptions
Esses tipos fazem parte da File System Access API e estão disponíveis em TypeScript via `lib.dom.d.ts` (TypeScript 4.x+). Se não estiverem disponíveis na versão atual, definir manualmente:
```typescript
interface SaveFilePickerOptions {
  suggestedName?: string
  types?: FilePickerAcceptType[]
}
interface OpenFilePickerOptions {
  types?: FilePickerAcceptType[]
  multiple?: boolean
}
```
