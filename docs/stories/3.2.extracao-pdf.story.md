Ready for Review

## Story

**As a** dev agent,
**I want** implementar o serviço de extração de texto e estrutura do PDF,
**so that** o pipeline de migração consiga identificar os campos presentes no documento preenchido do planetexpress.

## Acceptance Criteria

1. `PDFExtractor` recebe o path do PDF e retorna lista de `TextBlock` com campos detectados
2. Cada `TextBlock` contém: `text`, `x`, `y`, `width`, `height` (coordenadas normalizadas 0–1), `page`, `font_size`, `font_name`
3. Extração suporta PDFs multi-página (todos os blocos de todas as páginas são retornados)
4. Tabelas detectadas pelo pdfplumber são incluídas como TextBlocks com `block_type: "table"`
5. Rota `POST /api/jobs` integra fase 1 do pipeline: extrai PDF do job e emite SSE `step: "Extraindo PDF"`, `pct: 20`
6. Em caso de PDF corrompido ou ilegível, emite SSE `event: error` com mensagem descritiva e não quebra o servidor
7. `GET /api/health` continua funcionando após instalação de pdfplumber

### Model TextBlock

```python
# backend/models/text_block.py
from pydantic import BaseModel
from typing import Literal

class TextBlock(BaseModel):
    text: str
    x: float          # normalizado 0–1
    y: float          # normalizado 0–1
    width: float      # normalizado 0–1
    height: float     # normalizado 0–1
    page: int
    font_size: float = 0.0
    font_name: str = ""
    block_type: Literal["text", "table"] = "text"
```

### PDFExtractor — Referência

```python
# backend/services/pdf_extractor.py
import pdfplumber
from pathlib import Path
from models.text_block import TextBlock
from typing import List

class PDFExtractor:
    def extract(self, pdf_path: Path) -> List[TextBlock]:
        blocks: List[TextBlock] = []
        with pdfplumber.open(pdf_path) as pdf:
            for page_num, page in enumerate(pdf.pages, start=1):
                pw, ph = page.width, page.height
                words = page.extract_words(
                    extra_attrs=["fontname", "size"]
                )
                for w in words:
                    blocks.append(TextBlock(
                        text=w["text"],
                        x=w["x0"] / pw,
                        y=w["top"] / ph,
                        width=(w["x1"] - w["x0"]) / pw,
                        height=(w["bottom"] - w["top"]) / ph,
                        page=page_num,
                        font_size=w.get("size", 0),
                        font_name=w.get("fontname", ""),
                    ))
                # Tabelas
                for table in page.extract_tables():
                    text = str(table)
                    blocks.append(TextBlock(
                        text=text, x=0, y=0, width=1, height=0.1,
                        page=page_num, block_type="table"
                    ))
        return blocks
```
